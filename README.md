# Processing Bank Transfers

Учебный backend-проект на Go для обработки банковских переводов.

## Текущий этап

Реализованы базовая бизнес-логика и минимальный каркас проекта:

- точка входа API (`cmd/api/main.go`) с health-check `GET /health`;
- доменные модели `BankAccount` и `Transaction`;
- интерфейсы `Repository` и `Service`;
- in-memory реализации репозиториев;
- сервис с операциями:
  - создание счета,
  - получение баланса,
  - перевод с проверкой достаточности средств,
  - получение истории транзакций;
- unit-тесты для ключевых сценариев бизнес-логики.

## Запуск

```bash
go run ./cmd/api
```

Проверка health-check:

```bash
curl http://localhost:8080/health
```

## Тесты

```bash
go test ./...
```
